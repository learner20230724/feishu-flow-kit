给 schema handoff 用的一张中文 review 单页

这页把现有的 demo-table-schema-review-page.svg 包进一个更干净的独立页面，方便直接本地打开、截图，或者从 README、issue、setup note 里外链引用，不必先截 markdown 长页。

raw field-list → normalized schema → mapping draft → env review baseline + advanced fixture chains review-first，不是 live schema discovery

这页适合拿来干什么

当你只想要一张干净的 review surface，用于 README 链接、setup note、issue 评论或本地截图导出时，用这页最合适。它刻意比完整 handoff demo 更短，也更适合传播。

  • 比完整 handoff demo 更短
  • 比长 markdown 截图更便携
  • 更适合 review thread、issue 和 repo 资产引用

最值得盯的三类信号

Select 结构 optionCount 说明这列携带了真实 option metadata，不只是退化成纯文本字段。
Date / datetime 漂移 rawSemanticType=datetime 加上 dateFormatter，能让 reviewer 快速判断当前 date mode 会不会太弱。
Linked-record relation shape rawSemanticType=duplex_link 加上 linkedTableId，能在第一次 live write 之前把 relation shape 留在 review 面上。
