  • more portable than a long markdown capture
  • better for review threads and external repo assets

High-signal checks

Select shape optionCount shows the column is carrying real option metadata, not just a text fallback.
Date vs datetime drift rawSemanticType=datetime plus dateFormatter tells the reviewer whether date mode is too weak.
Linked-record relation shape rawSemanticType=duplex_link plus linkedTableId keeps relation review visible before first live writes.
