## What's New in v1.0.3

**ЁЯОБ Plugin Ecosystem** тАФ the biggest feature in this release. A fully extensible slash-command architecture with lifecycle hooks (`register`, `beforeProcess`, `handle`, `onCommandResult`, `afterProcess`), `PluginRegistry` for collision prevention, and error isolation so plugin failures never crash webhooks.
