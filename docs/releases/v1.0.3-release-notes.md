## What's New in v1.0.3

**🎁 Plugin Ecosystem** — the biggest feature in this release. A fully extensible slash-command architecture with lifecycle hooks (`register`, `beforeProcess`, `handle`, `onCommandResult`, `afterProcess`), `PluginRegistry` for collision prevention, and error isolation so plugin failures never crash webhooks.

### ✨ New in v1.0.3

| Category | Feature |
|----------|---------|
| **Plugin System** | `PluginRegistry`, lifecycle hooks, `loadPlugins()` from `FEISHU_PLUGINS` env var |

## ⚠️ Next Step — Requires Maintainer Action

To enable **@feishu/plugin-template npm publishing**, add `NPM_TOKEN` secret:
1. Go to **Settings → Secrets and variables → Actions → New repository secret**
2. Name: `NPM_TOKEN`
3. Value: Create an npmjs.com automation token with "Publish" permission for `@feishu` scope
4. Then go to **Actions → Publish @feishu/plugin-template to npm → Run workflow**
